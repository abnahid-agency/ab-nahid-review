# **App Name**: Ab Nahid Review Generator

## Core Features:

- Review Generation: Generate a positive Google review, including a keyword tool, for Ab Nahid Agency
- Prompt display: Display a user prompt requesting information relevant to the Google Review in a clear and concise format.
- Generated review display: Display the generated review in an editable text field.

## Style Guidelines:

- Primary color: Saturated blue (#4285F4) to evoke trust and reliability.
- Background color: Light blue (#E3F2FD), a desaturated version of the primary to create a calm background.
- Accent color: Yellow-orange (#FFCA28), for highlighting important actions and information.
- Font: 'Inter', a grotesque sans-serif for a modern and neutral look in both headlines and body text.
